Offers a simple GlassPane application for deactivating parts of the user interface. Deactivated contents can be blurred and/or grayed out. Furthermore, this package contains 'notification' classes which can display information on the current activity on top of the glass pane.

Core classes

BlurringGlassPane

As the class name gives away, this is a glasspane implementation which will blur the underlying content when visible. Can be used to block entire frames or single containers by using a JLayeredPane. @since 0.1
