Provides extensions to JFileChooser.

Core classes

ExtensionFileFilter

A simple FileFilter implemenation which relies on configuration rather than subclassing. There are subclasses available for the most frequent file types.

AccessoryManager

Allows installing accessory components into an existing JFileChooser. Unlike the built-in functionality, an AccessoryManager can be used to install multiple accessories at the same time and in different locations. @since 0.1
