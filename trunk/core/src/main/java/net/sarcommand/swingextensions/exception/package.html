Contains a set of classes designed for handling exceptions.

Core classes

Exception dialog

A dialog implementation for displaying exceptions in an application. Can be customized in order to provide additional user actions when reaction to such an error (such as submitting an error report).

AWTExceptionHandler

A subclass of Thread.UncaughtExceptionHandler which can be attached to the AWT event thread in order to intercept exceptions generated from within the user interface. @since 0.1
