Provides classes revolving around managing progress. This package includes both listener interfaces and adapters and swing components for displaying the progress of an activity.

Core Classes

JProgressIndicator

A swing component for indicating an indeterminate progress. It realizes the 'spinning wheel' known from numerous web 2.0 applications.

ProgressListener

An EventListener subclass designed to track the progress of an activity. Comes with its own event type and several adapter classes. @since 0.1
