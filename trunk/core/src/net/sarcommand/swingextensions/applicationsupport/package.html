Contains classes that handle the usual basic tasks for desktop applications. This includes utilities for accesing environment variables, window management and icon loading. Furthermore, a basic interpreter for application parameters is available.

Core classes

Application

A base class for a desktop applications entry point. This class is designed to be used as a singleton. It provides access to environment variables and offers basic parameter handling utilities.

ImageCache

Offers sophisticated mechanism for looking up, loading and caching image and icon resources within an application. This class is designed to be used as a singleton within an application. It provides configurable search paths for resources and a configurable error handling strategy.

WindowManager

For multi-window applications, this class helps providing a consistent behavior between frames. It handles window positions persistently and offers means to bring all application windows back into focus when one is selected. @since 0.1
